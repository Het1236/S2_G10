\vspace{0.3cm}

The project execution consists of major milestones (M1--M6):

\begin{itemize}
    \item \textbf{M1: Kickstart:}\\
    Team formation, area identification, background, motivation, and problem formulation

    \item \textbf{M2: Mathematical Modeling:}\\
    Mathematical modeling of the selected problem domain

    \item \textbf{M3: Coding:}\\
    Simulation and computation

    \item \textbf{M4: Inference:}\\
    Choose, understand, and code a randomized algorithm

    \item \textbf{M5: Randomized Algorithms:}\\
    Apply randomized algorithm to the domain problem and obtain results

    \item \textbf{M6: Derive Bounds, Analysis, and Final Deliverables Submission}
\end{itemize}
